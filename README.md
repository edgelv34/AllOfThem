# AllOfThem
주로 사용하는 기술들을 사용하여 만든 프로젝트

## 기술 스택
Clean Architecture: Domain, Data, Presentation(ui) 레이어 분리
</br>
MVI 패턴: Intent 기반 상태 관리
</br>
Jetpack Compose: UI 구현
</br>
Coroutine & Flow: 비동기 처리
</br>
Dagger Hilt: 의존성 주입
</br>
Room: 로컬 데이터베이스
</br>
Retrofit: API 통신
</br>
CameraX: 카메라 기능
</br>
Location: 위치 정보
</br>

(현재는 위 기술을 기반으로 진행 예정)
